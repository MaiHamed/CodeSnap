# 📸 CodeSnap

CodeSnap is a simple and aesthetic web tool that lets you **paste, format, preview, and export your code as beautiful images** — perfect for sharing on social media or documentation.

---

## ✨ Features

* 🎨 Multiple themes (Default, Pink, Ocean, Neon, Sunset, etc.)
* 🧠 Automatic syntax highlighting using **Highlight.js**
* 🧹 Code formatting with **Prettier**
* 📸 Export code as image using **html2canvas**
* ⚡ Live preview while typing
* 💻 Clean and responsive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom themes + variables)
* JavaScript (Vanilla JS)
* Highlight.js
* Prettier
* html2canvas

---

## 🚀 How to Use

1. Paste your code into the textarea
2. Choose a theme from the dropdown
3. Click **Format Code** (optional but recommended)
4. Click **Export Image**
5. Done 🎉
